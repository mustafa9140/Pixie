Introduction

Welcome to Pixie Multipurpose Website HTML5 Template

Created: February 2024

Author: Nexus Themes

Profile: https://themeforest.net/user/nexus-themes

Contact: Support

Pixie is a responsive website template. It is fully flexible user-friendly and responsive template that looks great on Desktops, Tablets, and Mobile Devices. This template is built with Boostrap 5, HTML5, CSS3, JQuery and SCSS. It has 1 Homepage, and 7 pre-built inner pages. It is built with well organized folder structure, clean and commented code.

If you want to create any kind of website like SaaS, StartUp, and Tech, then you are at the right place. Purchase Pixie Multipurpose Website HTML5 Template and build your own dream. We have used vs-code based build tools and scss variables-based modes. You can quickly change the colors, font sizes, etc. in style file. We already designed it and you can easily design your website just how you like it.

Quick Start

Download and unzip the latest package from ThemeForest

You can simply use the dist/ folder as everything is included in this package.
Customizing with VS Code

To custmize the SCSS present with the template, perform the following simple steps:

1. Download and install Visual Studio Code (if you dont have it installed)

2. Open the dist/ folder in Visual Studio Code

3. Install the VS-Code extension Live Sass Compiler

4. Finally, simply press the Watch Sass button from the status bar to turn on the live compilation and then click to Stop Watching Sass from the status bar to turn off live compilation.
